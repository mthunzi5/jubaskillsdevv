# Request Hub - Quick Start Guide

## For Staff

### Creating a Request
1. Navigate to **Request Hub** from the sidebar
2. Click **Create New Request**
3. Fill in the form:
   - **Title**: Clear, descriptive name (e.g., "November Timesheets")
   - **Description**: Detailed instructions for interns
   - **Request Type**: Select from dropdown (timesheet, ID, certificate, etc.)
   - **Deadline**: Optional - set due date and time
   - **Target Audience**: Choose who should respond
     - All Interns
     - Varsity Interns Only
     - TVET Interns Only
     - Specific Intern (select from dropdown)
   - **Document Settings**:
     - Check "Require Document Upload" if files needed
     - Set max documents (1-25)
     - Check "Include Text Field" if written response needed
     - Set text field label (e.g., "Explanation", "Comments")
4. Click **Create Request**

### Viewing & Managing Requests
- **Request List**: Shows all requests with completion stats
- **View Details**: Click on a request card to see:
  - All expected respondents
  - Who has/hasn't submitted
  - Submission status (pending, approved, rejected)
- **Toggle Active/Inactive**: Use play/pause button
- **Delete Request**: Use trash icon (warning: deletes all submissions)

### Reviewing Submissions
1. Go to request details page
2. Click on a submission to review
3. View uploaded documents and text responses
4. Add review notes (optional)
5. Click **Approve** or **Reject**

## For Interns

### Viewing Your Requests
1. Navigate to **My Requests** from the sidebar
2. You'll see all requests assigned to you:
   - Green border = Already submitted
   - Status badge shows: Pending Review, Approved, or Rejected
   - Red "Overdue" badge if past deadline

### Submitting a Response
1. Click on a request card
2. Read the requirements carefully
3. If documents required:
   - Click **Add File** to add upload fields
   - Select file
   - Enter a name for the document (optional but recommended)
   - Click **Add File** again for more files (up to limit)
   - Use trash icon to remove files
4. If text field required:
   - Type your response in the text box
5. Click **Submit**

### Updating Your Submission (Before Deadline)
1. Go to the request
2. Your previous submission will be shown
3. You can:
   - Delete existing documents (trash icon)
   - Add more documents (if under limit)
   - Update text response
4. Click **Update Submission**

### After Deadline
- You cannot submit or update after the deadline
- You can still view your submission and staff feedback

## Tips

### For Staff
- Use clear, specific request titles
- Provide detailed instructions in the description
- Set realistic deadlines (consider intern workload)
- Review submissions promptly
- Provide constructive feedback when rejecting
- Use "Varsity/TVET Only" to target specific groups
- Set appropriate max documents (timesheets might need 5, ID copy needs 1-2)

### For Interns
- Submit before the deadline (don't wait until last minute)
- Name your documents clearly (e.g., "Week 1 Timesheet", "ID Front", "ID Back")
- Follow the instructions carefully
- Check for staff feedback regularly
- If rejected, read the feedback and resubmit (if before deadline)
- Keep copies of all documents you submit

## Common Use Cases

### Monthly Timesheets
- **Type**: Timesheet
- **Target**: All Interns or by type
- **Max Documents**: 4-5 (one per week)
- **Deadline**: 5th of following month
- **Text Field**: Optional (for explanations of absences/overtime)

### ID Verification
- **Type**: ID Document
- **Target**: Specific intern or all
- **Max Documents**: 2 (front and back)
- **Text Field**: No
- **Deadline**: As needed

### Proof of Registration
- **Type**: Proof of Registration
- **Target**: Varsity Interns Only
- **Max Documents**: 3 (registration confirmation, fee receipt, etc.)
- **Text Field**: Optional (for special circumstances)
- **Deadline**: Start of semester

### Monthly Reports
- **Type**: Report
- **Target**: All Interns
- **Max Documents**: 2-3 (report + supporting docs)
- **Text Field**: Yes ("Summary of key achievements")
- **Deadline**: End of month

## Troubleshooting

### Staff Issues
**Q: Why can't I see any submissions?**
A: Interns may not have submitted yet, or the request might not be active.

**Q: Can I edit a request after creation?**
A: No, but you can deactivate it and create a new one.

**Q: What happens when I delete a request?**
A: All submissions and uploaded files are permanently deleted. Use with caution.

### Intern Issues
**Q: Why don't I see a request?**
A: It may not be targeted to you (check if it's for specific interns or different intern type).

**Q: I can't submit after the deadline**
A: Correct, submissions are locked after deadline. Contact staff if you need an extension.

**Q: I can't delete my uploaded document**
A: You can only delete before the deadline. After deadline, submissions are locked.

**Q: My submission shows "Pending Review" for a long time**
A: Staff needs to review it. Be patient, or follow up politely.
